These are some of the activities I made while studying Photoshop. Something to make clear before starting is that my teacher gave us instructions and tauch us methods we had to follow. Despite of this even if we used decided to use the images he gave us, we were the ones chosing which one of them we wanted to use and how to use it. This message is only to clarify that even it these work are parte of activities with rules and requirements, they are original and not a copy of what the teacher was doing during the class. Selective Correction

This was a very simple activity, we had to add contrast, color correction and change lighting to these backgrounds. We did that splitting the images in different parts so that we could make color correction depending on the main colors of each part. For example in the beach image we corrected individually the sea, the sand and the sky.
